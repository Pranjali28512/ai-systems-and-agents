# RAG vs AI Agents

## What is RAG?

RAG stands for Retrieval Augmented Generation.

It improves LLM responses by retrieving relevant information from a knowledge base before generating an answer.

Workflow:

User Question  
↓  
Retrieve documents from vector database  
↓  
Provide context to LLM  
↓  
Generate response

Used in:
- Knowledge chatbots
- Document Q&A systems
- Enterprise knowledge assistants

---

## What is an AI Agent?

AI agents go beyond RAG.

They can:

- Decide what actions to take
- Use tools
- Execute multi-step tasks

Example:

User asks:
"Find the best flights to Zurich and summarize the options."

An agent could:

1. Search flight APIs
2. Analyze results
3. Compare prices
4. Present best options

---

## Key Difference

RAG retrieves information.

Agents take actions.
